# GoF - Gang of four - банда четырех

# GRASP - General Responsibility Assignment Software Patterns - общие шаблоны распределения ответственностей

# Fundamental
- Property container - контейнер свойств
  . дает возможность добавлять свойства в уже созданный объект
- Delegation - делегирование
- Event channel - канал событий
  . список подписчиков, список издатей, событие
- Interface - интерфейс (абстракция к методам в классе)

# Creational(Generating) patterns - порождающие шаблоны 
  (Абстрагируем процесс инстанцирования, т.е. вместо конструкции языка new Circle() пишем CircleBig() )
  (Есть класс Circle, у него есть св-ва радиуса и положения x y)
  (Используем два вида объекта класса Circle - CircleBig CircleSmall)
  (Если нужно будет поменять радиус, то заменем его в одном месте)
- Singleton - одиночка
- Factory - фабрика
- Abstract factory - абстрактная фабрика
- Prototype - прообраз

# Structural
- Adapter/Wrapper - адаптер
- Bridge - мост
- Facade - фасад (простой интерфейс сложной библиотеки) 
           (интерфейс = заказ пиццы, библиотека = продукты, кухня, доставка, люди, налоги)
           (фасад не обязательно реализует весь возможный функционал)
           (классы в библиотеке не знают о фасаде)

# Behavior - поведенческие
- Memento - снимок, хранитель
- Observer - наблюдатель
- Visitor - посетитель
- State - состояние
- Iterator - итератор
- Mediator - посредник
